## How to run Locally:

### Server:
1. Directory `GetJson`
2. create `.env.development` file and add your own variable value included in `.env.example`
3. Run `yarn` to install all packages
4. Run `yarn dev` to run the server

### React:
1. Directory `GetJson/client`
2. create `.env.development` file and add your own variable value included in `.env.example`
3. Run `yarn` to install all packages
4. Run `yarn dev` to run the server
